## 🆘 Troubleshooting

**Issue**: Module not found
→ Run `npm install`

**Issue**: Port already in use
→ Kill process on that port

**Issue**: API key not working
→ Check it starts with `sk-ant-v01-`

**Issue**: CORS errors
→ Ensure backend CORS is enabled
